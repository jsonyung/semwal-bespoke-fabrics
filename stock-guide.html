Stock Guide

Hide, restore, add, and publish fabric catalog changes.

Open Catalog Download PDF

Daily Rule

Active fabrics live in the images/ folder. Out-of-stock fabrics live in archive/out-of-stock/. The website, PDF, thumbnails, and search data are rebuilt from the active images/ folder.

For normal office work, hide fabrics with the script. Do not delete the photo unless you are sure it will never be needed again.

Hide Out-Of-Stock Fabric

1. Open Terminal In This Project

Go to the project folder, usually:

cd /Users/sbtailor/StudioProjects/aa/semwal-bespoke-fabrics

2. Run The Hide Command

Replace the code with the fabric code you want to hide.

./mark-out-of-stock.sh I-440

For many fabrics, run one line per fabric:

./mark-out-of-stock.sh I-358
./mark-out-of-stock.sh I-366
./mark-out-of-stock.sh I-367

3. Approve Publish

When Terminal asks this question, type y and press Enter.

Commit and push these catalog changes now? [y/N]

Bring Fabric Back

Use this when the fabric is available again. It moves the photo back into images/ and rebuilds the website/PDF.

./restore-fabric.sh I-440

Add New Fabric

1. Add Photo

Put the new fabric photo in images/. Name it with the fabric code, for example I-463.jpg or PI-532.jpg.

2. Rebuild Catalog

./update-catalog.sh

3. Publish

Type y when it asks to commit and push. GitHub Pages can take a few minutes to refresh online.

Check Your Work

  • Search the code on the website. Hidden fabrics should show no result.
  • Open the PDF and search the code with Cmd+F or Ctrl+F.
  • If the online website looks old, hard refresh with Cmd+Shift+R on Mac or Ctrl+Shift+R on Windows.
