# Semwal Bespoke Fabrics Project Plan

## Stage 1 - Done

- Created a clean project folder.
- Copied all fabric images into `images/`.
- Kept each filename as the searchable fabric code.
- Generated `CATALOG.md` for GitHub search.
- Generated `catalog-data.json` for website search.
- Added `index.html` for a searchable browser catalog.
- Generated `semwal-bespoke-fabrics-catalog.pdf` for sharing and printing.

## Stage 2 - GitHub

1. Create a new GitHub repository named `semwal-bespoke-fabrics`.
2. Push this folder to that repository.
3. Open `CATALOG.md` on GitHub.
4. Search any code with browser search, like `I-440`.

## Stage 3 - GitHub Pages

1. In GitHub, open repository settings.
2. Go to Pages.
3. Select deploy from the `main` branch and `/root` folder.
4. Open the generated GitHub Pages link.
5. Use the search box in `index.html` to find fabrics by code.

Expected website URL:

`https://jsonyung.github.io/semwal-bespoke-fabrics/`
